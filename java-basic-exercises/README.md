# Java Basic Exercises

Esta pasta contém exercícios básicos de Java, organizados por tópicos e categorias para facilitar o estudo e prática.

## Estrutura

### 1. Arrays e Listas
- **CadastroNotas.java**: Exercício para cadastro e manipulação de notas.
- **ListaNomes.java**: Exercício de manipulação de listas de nomes.
- **MatrizNotas.java**: Exercício usando matrizes para armazenar notas.

### 2. Condições e Menus
- **DiaUtil.java**: Verifica se um dia da semana é útil ou fim de semana.
- **MenuSimples.java**: Exercício de menu interativo simples.
- **VerificadorIdade.java**: Verifica se a idade atende a critérios específicos.

### 3. Datas
- **ManipulaData.java**: Exercício de manipulação de datas em Java.

### 4. Exceções
- **TrataErrosAvancado.java**: Exercício avançado de tratamento de erros.
- **TrataExcecao.java**: Exercício de captura de exceções básicas.
- **TrataNumeroInvalido.java**: Validação de entrada de números.

### 5. Exemplos
- **Main.java**: Código exemplo demonstrando conceitos básicos de Java.

### 6. HackerRank
- **HackerRank1.java**
- **HackerRank2.java**
> Exercícios resolvidos do site HackerRank.

### 7. Jogos
- **AdivinhaNumero.java**: Jogo de adivinhação de números.
- **JogoAdivinhacaoPontos.java**: Jogo de adivinhação com sistema de pontuação.

### 8. Loops
- **TabelaMultiplicacao.java**
- **TabelaMultiplicacaoAninhada.java**
- **TabelaMultiplicacaoSimples.java**
> Exercícios de loops simples e aninhados, mostrando tabelas de multiplicação.

### 9. Strings
- **ConvertorString.java**
- **ManipulaString.java**
> Exercícios de manipulação de strings.
